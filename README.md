Proyek ini adalah *front-end* aplikasi manajemen data yang berfungsi sebagai antarmuka utama untuk sistem Pendataan Publikasi. Dibangun menggunakan *framework* **Vue.js**, aplikasi ini bertujuan untuk mendigitalisasi, mengorganisir, dan memfasilitasi akses cepat terhadap data publikasi melalui antarmuka pengguna yang reaktif dan efisien.

## ğŸŒŸ Fitur Utama Aplikasi

Proyek ini menyediakan kapabilitas penuh dalam pengelolaan data publikasi:

* **Manajemen Data (CRUD):** Fungsionalitas lengkap untuk membuat (Create), membaca (Read), memperbarui (Update), dan menghapus (Delete) entri data publikasi (e-Publication).
* **Pencarian dan Filter Lanjutan:** Pengguna dapat melakukan pencarian data secara spesifik dan menggunakan filter berdasarkan kriteria seperti judul, nama penulis, tahun publikasi, dan kategori/jenis publikasi.
* **Visualisasi Data:** Dashboard utama menampilkan statistik data publikasi yang *up-to-date* melalui komponen grafik dan ringkasan metrik.
* **Antarmuka Reaktif (SPA):** Menggunakan arsitektur Single Page Application (SPA) dari Vue.js, memastikan pengalaman pengguna yang mulus dan cepat tanpa *page reload* penuh.
